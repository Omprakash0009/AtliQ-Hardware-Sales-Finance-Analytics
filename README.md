# Brick & Mortar Business Analysis | Excel Project

## Project Overview

This project focuses on analyzing sales and financial performance data for AtliQ Hardware using Microsoft Excel, Power Query, and Power Pivot.

The objective of this project was to transform raw business data into actionable insights through ETL processes, data modeling, and interactive dashboards. The project demonstrates how Excel can be used as a business intelligence and analytics tool for reporting and decision-making. :contentReference[oaicite:0]{index=0}

---

# Problem Statement

AtliQ Hardware relied heavily on multiple Excel files for business reporting and analysis. The company wanted to streamline reporting processes, reduce dependency on scattered Excel files, and improve reporting efficiency using analytics and automation.

Business users required dynamic reports to analyze customer performance, market performance, and financial trends. The goal was to create professional reports that support data-driven business decisions.

The required reports included:
- Customer Performance Report
- Market Performance vs Target Report
- Sales Analytics Reports
- Finance Analytics Reports
- Profit & Loss Reports

---

# My Role

As a Data Analyst, my responsibilities included:

- Extracting, transforming, and loading business data
- Performing ETL operations using Power Query
- Cleaning and transforming raw datasets
- Building data models using Power Pivot
- Creating dashboards and reports
- Analyzing sales and financial performance
- Delivering actionable business insights
