# AtliQ Hardware Sales & Finance Analytics | Excel Project

## Project Overview

This project focuses on analyzing sales and financial performance data for AtliQ Hardware using Microsoft Excel, Power Query, and Power Pivot.

The objective of this project was to transform raw business data into meaningful insights through ETL processes, data modeling, and interactive reports. This project demonstrates how Excel can be used as a powerful business intelligence and analytics tool for reporting and decision-making.

The project includes customer performance analysis, market performance tracking, profit and loss analysis, and product-level insights. :contentReference[oaicite:0]{index=0}

---

# Problem Statement

AtliQ Hardware relied heavily on multiple Excel files for business reporting and analysis. The company wanted to streamline reporting processes, reduce manual work, and generate actionable insights from large datasets.

As a Data Analyst, the objective was to:
- Perform ETL (Extract, Transform, Load) operations
- Clean and transform raw data
- Build a scalable data model
- Create analytical reports and dashboards
- Provide insights to help business growth and decision-making

---

# My Role

As a Data Analyst, my responsibilities included:
- Extracting and transforming raw business data
- Performing ETL operations using Power Query
- Building relationships between tables using data modeling
- Creating dynamic reports and dashboards
- Analyzing sales and finance performance
- Delivering actionable business insights
